My personal website running on [Next.js](https://nextjs.org/)

## Built using

- [Next.js + Typescript](https://nextjs.org/)
- [MDX using `mdx-bundler`](https://github.com/kentcdodds/mdx-bundler/)
- [Tailwind CSS](https://tailwindcss.com/)

## Development

```bash
npm install
npm run dev
```
